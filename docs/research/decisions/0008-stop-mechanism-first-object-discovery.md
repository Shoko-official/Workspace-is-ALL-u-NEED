## Findings

### The report supplies an assembly, not a new mechanism

The report combines domain vocabulary adaptation, UniTok, LLM routing,
speculative decoding, hybrid attention/SSM layers, MoE, and serving
specialization. The cited works do not evaluate that system or establish its
aggregate claims.

The most important source mismatch is categorical. UniTok tokenizes
recommender-system items represented by semantic embeddings into discrete item
identifiers. It does not lexically segment prose, code, or mathematics and does
not contain the report's proposed code tokenizer. The genuine text-tokenization
branch is already occupied by domain vocabulary customization, dynamic
tokenization, evolving vocabularies, and tokenizer transfer.

For reversible text tokenizers, the exact map `T_b o D_a` transcodes a record
from writer tokenizer `a` to reader tokenizer `b`. A matched comparator can
therefore reproduce the same reader input and compute decision. For a
non-injective learned codebook, lost information cannot be recovered by more
compute. Model-specific latents, embeddings, or KV payloads instead reopen the
already stopped representation-migration and writer-reader objects.

### Shared computation cannot create the requested separation

Let a joint randomized circuit `J(h,r)` return memory and compute actions
`(m,c)` with work `W`, depth `D`, and fast memory `S`. With common public input
and randomness, two parallel copies reproduce the exact joint law using at
most `2W` work, aggregate fast memory `2S`, depth `D`, and no message.
Sequential copies instead use fast memory `S` and depth `2D`. For a finite
output alphabet, a single execution followed by transmission of one output
gives a second compiler.

Any superconstant separation therefore requires private or late observations,
private randomness, isolated memories, restricted fanout, or a bounded
message. Those restrictions are communication complexity, channel synthesis,
cell-probe, circuit, or pebbling problems already covered by M1e or generic
theory. TriRoute also directly occupies the recent empirical shared-controller
theme, although its July 2026 preprint remains unreviewed and unreplicated.

### The remaining blind spots are occupied classes

- Evidence inflation recycles provenance, contamination, and consolidation
  feedback and has direct LLM-memory demonstrations.
- Cross-model semantic ABI recycles runtime drift and decision-preserving state
  migration; Rosetta Memory and AFTER are direct positive records.
- Durable experience displacing reasoning compute is stated by SpeedupLLM.
- Non-malleable origin-bound authority is stated by TMA-NM and composed from
  established information-flow and capability mechanisms.
- Secret-dependent depth, stopping-time, and latency traces are the established
  output-plus-runtime timing-privacy object when the full interaction for a
  fixed adversary is treated as one randomized program. Broader address,
  branch, call, or action channels are outside this source-backed disposition.
- A universal executable-write gate is ordinary undecidability and program
  verification; SEVerA supplies a restricted constructive regime.
- Strategic memory contribution with costly audits is online mechanism design
  for information acquisition and adaptive costly verification.

### Complete no-recycling audit

The M1f register enumerates all 64 previous objects. Each potential overlap is
marked before admission. The primitives that are genuinely outside the exact
earlier labels still fail direct collision, generic reduction, label
substitution, or a constructive compiler. This distinction prevents the
crosswalk itself from being misused as an exhaustiveness claim.

## Decision

Propose global `STOP` for all nine M1f mechanism-first attempts.

- No object is admitted.
- No object is `DISTINCT_CANDIDATE`.
- No theorem, estimand, invariant, capability, complexity separation, or
  transportable law remains specific to governed persistent LLM state plus
  adaptive computation.
- No model, benchmark, implementation, training run, experiment, paid compute,
  proof project, manuscript, or submission is authorized.
- The recovered report is now an available input, but it does not establish an
  article route.

The program-level direction-setting objective remains active. This decision is
a negative originality gate, not the requested article.

## Anti-rewrite determination

The following are explicitly insufficient for a future article:

- relabeling domain vocabulary adaptation as tokenization "by mission";
- treating recommendation item identifiers as lexical text tokens;
- multiplying gains reported on different tasks, models, baselines, and cost
  regimes into one system-level claim;
- presenting a shared neural trunk or constant-factor common-subexpression
  saving as an intrinsic state-compute separation;
- weakening the factorized comparator by hiding public inputs, randomness,
  activations, codecs, or a shared frontend;
- renaming timing privacy, IFC, mechanism design, schema migration, channel
  synthesis, pebbling, or verification as a workspace theory;
- converting the recovered report or the nine negative objects into a survey,
  taxonomy, manifesto, roadmap, or architecture proposal; or
- running an experiment in order to manufacture novelty after the object gate
  has failed.

## Alternatives considered

- `GO` on mission-conditioned tokenization was rejected by the UniTok category
  error, exact transcoding, dynamic-tokenizer prior art, and M1C-C09.
- `GO` on shared computation was rejected by the exact two-copy compiler and
  the single-copy-plus-message compiler.
- `GO` on origin-bound authority was rejected by the direct TMA-NM collision.
- `GO` on trace privacy was rejected because declared depth, stopping time, and
  latency are observable runtime in timing privacy.
- `GO` on executable persistent skills was rejected by generic undecidability
  and the existing restricted SEVerA construction.
- `GO` on strategic records was rejected by online information-acquisition and
  costly-audit mechanism design.
- A minimum empirical experiment was rejected because none of these failures
  is empirical underidentification.
