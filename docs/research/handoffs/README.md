# Research handoffs

Each handoff must contain:

1. objective;
2. scope and exclusions;
3. hypotheses or questions tested;
4. inputs and source versions;
5. work completed;
6. evidence with precise locators;
7. uncertainties and contradictions;
8. files modified;
9. verification executed;
10. recommended decision;
11. next authorized action;
12. authoring functional role and independent reviewing role.

A handoff is evidence for a gate only after a distinct functional role has reviewed it.
